# Java Banking Application  
  
## Description  
A simple Java console-based banking application developed using Core Java concepts.  
  
## Features  
- Create bank account  
- Deposit money  
- Withdraw money  
- Check balance  
  
## Technologies Used  
- Core Java  
- OOP Concepts  
- Collections  
- Exception Handling (Basic)
